Features

Everything you need to run a job board

From job posting to monetization, Modern Job Board handles it all while giving you complete control.

Frontend Job Submission

Employers post jobs directly from your site with an intuitive form. Supports custom fields, application methods, and company profiles.

AJAX Filtering

Lightning-fast job search with filters for keywords, location, category, and job type. No page reloads needed.

Candidate System

Complete candidate management with profiles, secure resume storage, and one-click applications using stored CVs.

WooCommerce Monetization

Multiple revenue streams: pay-per-post, job credit packages, CV unlocks, and time-based access passes.

Google Maps Integration

Visual location display on job pages helps candidates understand where opportunities are located.

SEO Ready

Automatic Schema.org JobPosting structured data for optimal visibility in Google for Jobs results.

Auto Expiration

Jobs automatically expire after your configured duration. Background cron jobs handle cleanup automatically.

Custom Fields Builder

Create unlimited custom fields for jobs and applications through a simple admin UI. No code required.

CSV Import/Export

Bulk import job listings or export your data for analysis. Perfect for migrations and reporting.

Built for extensibility

REST API, XML feeds, and extensive hooks make integration straightforward.

REST API

Access job listings programmatically with the built-in JSON API. Perfect for mobile apps, external integrations, or headless WordPress setups.

XML Job Feed

Syndicate your listings to Indeed, Google Jobs, and other aggregators with the standardized feed at /feed/job-listings

Actions & Filters

Dozens of hooks let you customize job queries, form submissions, email notifications, and more without touching core plugin files.

// REST API Endpoint
GET /wp-json/mjb/v1/jobs

// Response
[
  "id": 123,
  "title": "Senior Developer",
  "company": "Acme Inc",
  "location": "Remote"
]

// Filter job queries
add_filter('mjb_job_query', function($args) {
  $args['orderby'] = 'rand';
  return $args;
});
